# 🚀 ContentFlow AI — Agentic Content Generator

ContentFlow AI is a modern, single-page web workspace designed for creators, copywriters, and marketers. It features an interactive chatbot assistant that collaborates with you to generate high-converting copy across multiple formats, instantly rendering the final text in a polished, readable live preview window.

Powered securely by a local Node.js proxy utilizing the **Google Gemini API**, it optimizes workflows while keeping your API credentials completely safe from frontend exposure.

---

## ✨ Features

- 🎭 **7 Contextual Content Types:** Instantly switch modes for Social Media Captions, Blog Posts, Product Descriptions, Email Templates, Video Scripts, Landing Page Copy, and Tweet Threads.
- ⚡ **Agentic Chat Workflow:** Interactive AI companion that prompts you for target audience details, goals, and tone before building production-ready outputs.
- 👁️ **Isolated Live Preview:** Separates conversational text from raw copy content into a beautifully styled, dedicated user-facing preview canvas.
- 📋 **One-Click Copy:** Quick-action clipboard copying complete with contextual button animations.
- 🛡️ **Secure Backend Middleware:** Built on Express to process data handling securely, keeping server secrets concealed.

---

## 🛠️ Tech Stack

- **Frontend:** Pure HTML5, Semantic CSS3 Grid layout, and Native Vanilla JavaScript.
- **Backend:** Node.js, Express framework, and Cross-Origin Resource Sharing (`cors`).
- **AI Integration:** Official Google Gen AI SDK (`@google/genai`) configured for optimized prompt instruction sets.
